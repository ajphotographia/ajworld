# AJ World — Open Source AI Prompt Workflow Kit

AJ World is an open-source AI prompt workflow kit for photographers, videographers, AI content creators, and prompt engineers.

The project provides reusable prompt templates, JSON structures, creative direction frameworks, and production-ready AI image/video workflows for creators who need consistent results across AI tools.

## What this project includes

- AI image generation prompt templates
- Face consistency prompt frameworks
- Outfit replacement prompt structures
- Character reference workflows
- Cinematic portrait prompt systems
- Instagram caption and content prompt templates
- JSON prompt templates for structured AI generation
- Creator-friendly documentation and examples

## Who this is for

This project is built for:

- AI content creators
- Photographers
- Videographers
- Prompt engineers
- Social media creators
- Creative agencies
- Designers experimenting with AI workflows

## Project goal

The goal is to help creators build repeatable, organized, and high-quality AI content workflows instead of writing random prompts every time.

## Example use cases

- Creating consistent AI portraits
- Maintaining face identity across AI images
- Changing outfits while preserving character identity
- Generating cinematic fantasy portraits
- Structuring JSON prompts for AI image tools
- Building repeatable client-ready creative workflows

## Status

This project is actively maintained and will continue to grow with new prompt templates, examples, documentation, and practical creator tools.

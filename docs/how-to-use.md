# How to Use AJ World

AJ World is a collection of open-source AI prompt workflows for creators.

## Step 1: Choose a prompt type

Select the prompt that matches your task:

- Face consistency
- Outfit change
- Fantasy portrait
- JSON prompt structure
- Social media content workflow

## Step 2: Upload your reference image

Use a clear image with good lighting. For face consistency, the reference image should show the face clearly.

## Step 3: Copy the prompt

Copy the prompt template and replace the placeholder details with your own subject, outfit, background, style, or camera direction.

## Step 4: Add strict identity instructions

For realistic results, always include identity protection instructions such as:

- Keep the same face
- Preserve exact facial features
- Do not redesign the subject
- Do not change hairstyle
- Keep real skin texture

## Step 5: Generate and refine

After generating the image, improve the result by giving specific correction prompts.

Example:

"Keep the same face as the uploaded reference image. Do not change the nose, eyebrows, hairstyle, or face shape. Make the image sharper and more realistic without changing identity."

## Goal

The goal of this project is to help creators build repeatable AI workflows instead of writing random prompts every time.
